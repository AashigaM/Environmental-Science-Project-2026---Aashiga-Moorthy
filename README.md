# Sunnyvale Real-Time AQI and Weather Tracker **Live Demo** : https://comforting-cascaron-6b9a16.netlify.app/

## Project Overview
This web application is a real-time evironmental dashboard deigned to track, analyze, and display the Air Quality Index (AQI) and current weather metric s for Sunnyvale, California. Created to address public health concerns surrounding fluctuating air quality in the Bay Area, the application automates data retireval to update every 10 minutes. It translates complex atmospheric data into clear, actionable health advisories, helping local residents make informed decisions about outdoor activities.

## Development and Architecture
The project is built on a modern frontend stack utlizing HTML5, CSS3, and JavaScript, compiled using the Vite build took. The application integrates third-plarty Restful APIs to fetch live climate and particulate matter data, dynamically updating the DOM to show real-time changes requiring a manual page refresh. It is continuously devloped and hosted via Netify, utilizing a version-controlled workflow linked directly to GitHUb.
